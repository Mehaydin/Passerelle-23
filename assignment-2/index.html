This will render it to look like a top level heading, but it has no semantic value, so it will not get any extra benefits as described above. It is therefore a good idea to use the right HTML element for the right job.

HTML should be coded to represent the data that will be populated and not based on its default presentation styling. Presentation (how it should look), is the sole responsibility of CSS.

Semantics in HTML
  • Search engines will consider its contents as important keywords to influence the page's search rankings (see SEO )
  • Screen readers can use it as a signpost to help visually impaired users navigate a page
  • Finding blocks of meaningful code is significantly easier than searching through endless divs with or without semantic or namespaced classes
  • Suggests to the developer the type of data that will be populated
  • Semantic naming mirrors proper custom element/component naming

SEO (Search Engine Optimization) is the process of making a website more visible in search results, also terned improving search rankings.

Search engines crawl the web, following links from page to page, and index the content found. When you search, the search engine displays the indexed content. Crawlers follow rules. If you follow those rules closely when doing SEO for a website, you give the site the best changes of showing up among the first results, increasing traffic and possibly reveneue (for ecommere and ads).

Search engines give some guidelines for SEO, but big search engines keep result ranking as atrade secret.SEO combines official search engine guidelines, empirial knowledge, and theoretical knowledge from science papers or patents.
